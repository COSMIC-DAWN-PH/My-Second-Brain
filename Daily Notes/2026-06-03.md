#### ⏰ 学习路线（三阶段 · 约 110 min）

##### Phase A · 硬件基础层（~25 min）

> 先搞清楚"原子在哪里、怎么被抓住的"——这是后面一切的物理载体

- [ ] **超精细能级 qubit 编码**（跨库关联）
  - 回顾已学内容：自旋、超精细结构、Zeeman 效应（你已学过）
  - 核心理解：$^{87}\text{Rb}$ 中 $\vert 0\rangle = \vert F{=}1, m_F{=}0\rangle$，$\vert 1\rangle = \vert F{=}2, m_F{=}0\rangle$，跃迁 $\sim 6.8\,\text{GHz}$
  - 为什么选超精细基态？→ 相干时间长（秒量级）、抗电场噪声
  - 关键桥梁：$\vert 1\rangle$ 到里德堡态 $\vert r\rangle$ 用**双光子跃迁**驱动
- [ ] **光镊阵列**：精读 [[Optical-Tweezer-Arrays]]
  - §1 物理直觉：光学梯度力 $U_{\text{dipole}} \propto -\alpha(\omega) I(\mathbf{r})$，红失谐 → 原子被吸引到焦点
  - §2 从单个光镊到阵列：SLM（空间光调制器）和 AOD（声光偏转器）
  - 重点：光镊阵列是**可扩展到千比特**的硬件基础

##### Phase B · 核心机制层（~60 min）⭐ 今天重点

> 里德堡阻塞 + CZ 门 + 时间最优脉冲 + 暗态抑制

**B1 · 里德堡阻塞机制（~30 min）**

- [ ] 精读 [[Rydberg-Blockade]]，按以下顺序：
  1. §1 物理直觉：里德堡态（$n \sim 70$）→ 轨道 $\sim n^2$ → 相互作用 $V \propto n^{11}$
  2. §2 数学：$V_{12} = C_6/R^6$，阻塞条件 $V_{12} \gg \Omega$，$R_b = (C_6/\Omega)^{1/6}$
  3. §3 阻塞实现 CZ 门的脉冲序列
  4. §4 定量：$P_{\text{error}} \sim (\Omega/V)^2$
- [ ] 看讲义 [[2023-parallel-gates-handout#4. Rydberg 态与 Rydberg blockade 物理]] 的 Python 可视化图

**B2 · CZ 门 + 时间最优控制（~20 min）**

- [ ] 精读 [[CZ-Gate]] §5 里德堡阻塞实现的四步流程

- [ ] 精读 [[2023-parallel-gates-handout#6. 核心创新一：时间最优单脉冲门]]
  - 矩形脉冲问题 → DRAG 脉冲：$\omega(t) = \omega_0 + \alpha \frac{d}{dt}(\Omega(t)/\Omega_{\max})$
  - 门时间从 $\sim 1\,\mu\text{s}$ 优化到 $\sim 300\,\text{ns}$

**B3 · 暗态抑制散射（~10 min）**

  - 散射概率从 $\sim 10^{-3}$ 降到 $\sim 10^{-5}$
