**3B · 第一个模拟：单 qubit Rabi 振荡（~15 min）**

- [ ] 用 QuTiP 模拟单 qubit 在常数驱动下的 Rabi 振荡
- [ ] 对照 [[Rabi-Flopping]] 中的 $P(t)=\sin^2(\Omega t/2)$ 公式
- [ ] 画出布居数随时间变化曲线

```python
# 示例框架（仅供参考）
import qutip as qt
import numpy as np

# 定义能级
g = qt.basis(2, 0)  # |0> ground state
e = qt.basis(2, 1)  # |1> excited state

# 定义算符
sz = qt.sigmaz()
sx = qt.sigmax()

# Hamiltonian (rotating frame)
Omega = 2 * np.pi * 1.0  # Rabi frequency (MHz)
H = (Omega / 2) * sx  # resonant drive

# 初始态
psi0 = g

# 时间列表
tlist = np.linspace(0, 10, 200)

# 求解 Schrodinger equation
result = qt.sesolve(H, psi0, tlist, [qt.ket2dm(e)])

# Plot
import matplotlib.pyplot as plt
plt.plot(tlist, result.expect[0])
plt.xlabel('Time (us)')
plt.ylabel('P(|1>)')
plt.title('Rabi Oscillation - QuTiP Simulation')
plt.grid(alpha=0.3, ls=':')
plt.show()
```

**原子与能级：**

| 参数 | 值 | 来源 |

**脉冲与相互作用：**

| 参数 | 符号 | 值 | 说明 |
|------|------|-----|------|
| 有效 Rabi 频率 | $\Omega$ | $2\pi \times 3.5$ MHz | 双光子有效耦合 |

| CZ 门总时间 | $2\tau$ | $\approx 2.732\pi/\Omega \approx 0.4\,\mu$s | 两脉冲方案 |
| Toffoli 远端相互作用 | $V_{\text{edge}}$ | $2\pi \times 0.4$ MHz | 三原子边缘对 |

**保真度目标（论文结果）：**

| 指标 | 值 | 说明 |
|------|-----|------|
| Bell 态保真度 | $F \geq 95.0(2)\%$ | 含 SPAM 误差 |
| CZ 门保真度 | $F_c \geq 97.4(3)\%$ | SPAM 修正后 |
| CNOT 真值表保真度 | $F_c^{\text{CNOT}} \geq 96.5(3)\%$ | SPAM 修正后 |
| Toffoli 保真度 | $F_c^{\text{Toff}} \geq 87.0(4)\%$ | SPAM 修正后 |

**误差来源（论文指出）：**

- 有限原子温度 → 多普勒展宽
- 里德堡态激光散射 → 非弹性退相干
- SPAM 误差：每原子 $\sim 1.2(1)\%$

> [!info] CZ 门物理原理
> 两个全局 Rydberg 脉冲（长度 $\tau$，失谐 $\Delta$，第二脉冲加相位跳变 $\xi$）驱动 $\vert 1\rangle \leftrightarrow \vert r\rangle$ 跃迁。$\vert 11\rangle$ 以 $\sqrt{2}\Omega$ 集体 Rabi 频率振荡，$\vert 01\rangle$ 以 $\Omega$ 振荡。选择 $\Delta \approx 0.377\Omega$ 使得两个系统的动态相位满足 $\phi_{11} = 2\phi_{01} - \pi$，实现 CZ 门。总门时间 $2\tau \approx 2.732\pi/\Omega$，比传统顺序方案 $4\pi/\Omega$ **快 1.46 倍**。

**路径一：脉冲优化假说**

- 用 GRAPE 最优控制算法设计脉冲，尝试提升保真度 0.5%-1%
- 调整双光子失谐量，抑制中间态泄漏
- 要求：提出明确可验证的假说（如"将失谐设为 $+2\pi \times 1$ MHz 可提升保真度到 98.0%"），再模拟验证

**路径二：差异复现 Levine→Evered（推荐起步）**

- 对比 Levine 2019（97.4%）与 Evered 2023（99.5%）的改进
- 找出每项技术改进（脉冲形状、原子间距、局域寻址等）各自贡献的保真度提升
- 在模拟中逐步替换参数，量化每步改进

**路径三：里德堡反阻塞（Antiblockade）机制（最有潜力）**

- 反阻塞 = 利用相互作用将双激发态拉入共振 → 快速纠缠
- 文献参考：Su et al. (PRA, 2024) 反阻塞 CZ 门方案
- 在现有模拟框架中实现反阻塞 Hamiltonian，推导共振条件
- 尝试结合反阻塞 + 脉冲优化

**路径四：系统级架构优化**

- 串扰分析：三原子线性排列中，中间原子同时参与左右门的串扰效应
- 温度效应：从几十 μK 降到几 μK，定量评估保真度提升 vs 降温时间开销
