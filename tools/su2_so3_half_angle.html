Single-Qubit Geometry · SU(2) → SO(3)

为什么 Bloch 球转 θ，ket 里却是 θ/2？

这不是“少转了一半”，而是两种描述对象不同：Bloch 球上的箭头描述可观测的物理方向； Hilbert space 中的 ket 描述两个复振幅。振幅取平方才变成概率，所以 ket 的角度自然写成 half-angle。
90° = π/2

核心结论

|ψ⟩ = cos(θ/2)|0⟩ + eiφ sin(θ/2)|1⟩

Bloch 球的极角是 θ；ket 中两个振幅的“分配角”是 θ/2。

拖动滑块观察

当 θ 从 0 变到 π，Bloch 箭头从北极走到南极；同时 ket 只从 |0⟩ 方向走到 |1⟩ 方向，也就是振幅角从 0 走到 π/2。

为什么不能写 cosθ 与 sinθ？

  • Bloch 球转 θ = π 时，状态应从 |0⟩ 到 |1⟩。
  • 用 half-angle：cos(π/2)=0，sin(π/2)=1，正好得到 |1⟩。
  • 如果误写 cosπ 与 sinπ，会得到 −|0⟩，物理上仍在北极，方向完全错了。

和旋转门的联系

Rn(θ)=exp[-i θ(n·σ)/2]

Pauli matrix 的本征值是 ±1，所以两个本征分量获得相反相位 e∓iθ/2；它们的相对相位差才是 Bloch 球上看到的 θ。
